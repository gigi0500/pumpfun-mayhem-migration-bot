An automated migration bot that monitors PumpFun tokens approaching their bonding curve completion and executes migration to Raydium DEX automatically. Capture opportunities during the critical migration phase.

## Features

- **Migration Detection**: Monitor tokens approaching bonding curve completion
- **Automatic Execution**: Execute migration transactions automatically
- **Timing Optimization**: Enter at optimal points before/during migration
- **Liquidity Monitoring**: Track liquidity changes during migration
- **Multi-Token Support**: Monitor multiple tokens simultaneously
- **Risk Management**: Configurable entry/exit strategies
- **Performance Tracking**: Track migration success rates
- **Token2022 Support**: Full support for Token2022 tokens during migration
- **Raydium Integration**: Seamless migration to Raydium DEX with optimized liquidity provision

## Integrations

### Token2022 Support
This bot fully supports migrating tokens using Solana's Token2022 standard:
- **Transfer Fees**: Handle tokens with transfer fees during migration
- **Confidential Transfers**: Support migration of privacy-enhanced tokens
- **Transfer Hooks**: Preserve custom transfer logic during migration
- **Metadata Extensions**: Maintain enhanced metadata through migration
- **Permanent Delegate**: Handle tokens with permanent delegate authority

### Raydium Integration
Native integration with Raydium DEX provides:
- **Seamless Migration**: Automated migration from PumpFun to Raydium DEX
- **Liquidity Provision**: Optimal liquidity provision during migration
- **Pool Creation**: Automatic Raydium pool creation and configuration
- **Price Discovery**: Smooth price discovery during migration transition
- **Multi-Pool Support**: Support for migrating to multiple Raydium pools

## Installation

```bash
git clone https://github.com/gigi0500/pumpfun-mayhem-migration-bot.git
cd pumpfun-mayhem-migration-bot
npm install
```

## Configuration

1. Copy `.env.example` to `.env`
2. Configure your wallet private key
3. Set your RPC endpoint
4. Configure migration detection parameters
5. Set entry/exit strategies
6. Configure risk management settings

## Usage

```bash
npm start
```
